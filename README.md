# TODO server

Todo backend.
NestJS + Mongoose + mongoDB + Swagger

### Configuration
.env
```
PORT=3000
MODE=DEV
SECRET=JWTSecret
JWT_EXPIRE=84600
DB_URI=MongoDBURL
```

### Getting Started
```
npm install
npm run start:dev
```
Swagger dest: http://localhost:3000/docs
